# JPG File Converter v1.1
JPG File Converter is a simple and user-friendly Windows desktop tool designed to quickly convert JPG and JPEG images into popular formats such as PNG, BMP, GIF, ICO, WEBP, JPG, and JPEG. It supports batch conversion, custom output folder selection, progress tracking, and a clean modern interface, making it useful for organizing photos, preparing website images, creating project graphics, or converting large image collections with minimal effort.

Trial version includes 5 free conversions. Upgrade to Pro for unlimited conversions.
